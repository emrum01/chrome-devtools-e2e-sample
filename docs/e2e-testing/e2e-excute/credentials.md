# 認証情報ポリシー

テスト実行に必要なアカウント情報の取得・保持方針をまとめています。社内向けの情報取り扱いルールを遵守してください。

## 取得対象
- `test-user@example.com` のパスワード
- `admin-user@example.com` のパスワード
- その他、実行するテストケースで要求されるテストアカウント情報

## 取得手順
1. テスト開始前に必要なアカウント情報をまとめて取得する。
2. セキュアストレージから取り出す場合は、アクセスログを残し適切に管理する。
3. 取得した情報はセッション中に再利用し、不要な再取得を避ける。

## 保持と取り扱い
- 認証情報はテストセッション中のみ保持し、終了後は速やかに破棄する。
- 社内のセキュリティポリシーに従い、共有チャンネルやログへ平文で出力しない。
- 外部への持ち出しや、用途外での利用を禁止する。
