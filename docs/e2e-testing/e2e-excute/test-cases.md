# テストケース索引

利用可能な Gherkin テストケースの概要と前提条件をまとめています。Runbook と併用し、実行対象を選定してください。

| テスト名 | 説明 | 前提条件 | 実行ファイル |
|----------|------|----------|--------------|
| ユーザー認証テスト | ログイン処理とユーザー ID 抽出テスト | • テストユーザーアカウントが存在する | [`features/ユーザー認証.feature`](features/ユーザー認証.feature) |
| 商品検索テスト | 商品の検索とフィルタリングテスト | • ユーザーがログインしている | [`features/商品検索.feature`](features/商品検索.feature) |
| 管理者ユーザー管理テスト | 管理者パネルからのユーザー役割変更テスト | • 管理者権限でログインしている<br>• 対象ユーザーアカウントが存在する | [`features/管理者ユーザー管理.feature`](features/管理者ユーザー管理.feature) |
| ショッピングカートテスト | 商品のカート追加とチェックアウトテスト | • ユーザーがログインしている<br>• 商品が利用可能 | [`features/ショッピングカート.feature`](features/ショッピングカート.feature) |
