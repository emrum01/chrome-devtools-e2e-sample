# 環境一覧

テスト実行時に利用可能な環境 URL と用途をまとめています。Runbook で対象環境を選定する際の参考にしてください。

| 環境 | アプリケーションタイプ | URL | 説明 |
|------|------------------------|-----|------|
| ローカル | メインアプリ | http://localhost:3000 | ローカル開発メインアプリケーション |
| 開発 | メインアプリ | https://dev.example.com | 開発環境 |
| ステージング | メインアプリ | https://staging.example.com | ステージング環境 |
